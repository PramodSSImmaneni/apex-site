Community

Mailing Lists

The following mailing lists can be used to get help with Apex.

  • dev@apex.incubator.apache.org is for people who want to contribute code to Apex. subscribe, unsubscribe, archives
  • users@apex.incubator.apache.org is a discussion forum for people who want to build apps and/or solutions using Apex. subscribe, unsubscribe, archives
  • commits@apex.incubator.apache.org is for commit messages and patches to Apex. subscribe, unsubscribe, archives

Contributing

To learn more about how we use Github to accept contributions to the project, check out the contributing guidelines.

The Apex Project is made up of two repositories:

  • Apex Core - The core of the Apex platform.
  • Apex Malhar - Community-driven set of open-source "operators" and utilities for use in your Apex applications.

Apache Apex is an effort undergoing incubation at The Apache Software Foundation (ASF), sponsored by the Apache Incubator. Incubation is required of all newly accepted projects until a further review indicates that the infrastructure, communications, and decision making process have stabilized in a manner consistent with other successful ASF projects. While incubation status is not necessarily a reflection of the completeness or stability of the code, it does indicate that the project has yet to be fully endorsed by the ASF.
